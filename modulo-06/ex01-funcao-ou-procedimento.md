# Exercício 1 - Função ou Procedimento?

1. **Str()**
   - **Classificação:** Função
   - **Justificativa:** Retorna um valor numérico convertido em string.

2. **QOut()**
   - **Classificação:** Procedimento
   - **Justificativa:** Apenas executa a ação de exibir texto na tela, sem retornar um dado para o programa.

3. **Date()**
   - **Classificação:** Função
   - **Justificativa:** Retorna um dado contendo a data atual do sistema.

4. **Len()**
   - **Classificação:** Função
   - **Justificativa:** Retorna um número correspondente ao tamanho de uma string ou de um array.

5. **Upper()**
   - **Classificação:** Função
   - **Justificativa:** Retorna a versão em maiúsculas da string fornecida.

6. **ClearScreen()**
   - **Classificação:** Procedimento
   - **Justificativa:** Apenas executa a ação visual de limpar a tela.
